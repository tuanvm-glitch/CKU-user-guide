# Rename a Delivery Method

Use this rule to change how a shipping method is labeled at checkout.

## When to Use

- Add estimated delivery time ("Standard" → "Standard Shipping (3–5 days)")
- Clarify method names for buyers ("Express" → "Next-Day Delivery")
- Localize names for specific markets

## Setup

1. Go to **Checkout Customization** → **Delivery** tab
2. Click **Create rule** → select **Change name of delivery method**
3. Set a **Rule name** and **Status** (Active / Inactive)
4. Configure **Trigger Conditions** (optional)
5. In the **Rule Action** section, add name mappings:
   - **Original name** — The exact name as it appears in Shopify
   - **New name** — The name buyers will see at checkout
6. Add multiple mappings if needed
7. Click **Save**
