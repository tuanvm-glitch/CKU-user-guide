# Reorder Delivery Methods

Use this rule to control the order in which shipping options appear at checkout.

## When to Use

- Move free shipping to the top to increase its selection rate
- Surface the cheapest or fastest option first
- Promote a specific carrier for certain order types

## Setup

1. Go to **Checkout Customization** → **Delivery** tab
2. Click **Create rule** → select **Reorder delivery method**
3. Set a **Rule name** and **Status** (Active / Inactive)
4. Configure **Trigger Conditions** (optional)
5. In the **Rule Action** section:
   - Type the delivery method name and click **Add**
   - Drag and drop the items to set the desired order
   - Methods listed here will appear first; unlisted methods follow in their default order
6. Click **Save**
