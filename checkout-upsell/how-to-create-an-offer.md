# How to Create an Offer

{% hint style="warning" %}
**Shopify Plus required.** You also need to add the AOV.AI block to your checkout theme before offers will appear. See the Installation page in Getting Started.
{% endhint %}

## Create a new offer

1. In the app, go to **Offers** in the left menu
2. Click **Create offer**
3. Select **Checkout upsell** and click **Select**

https://prnt.sc/nBlGREUCYT7z

4. Complete the setup steps:
   - **Step 1** — Set trigger conditions (or choose "No condition" to show to everyone)
   - **Step 2** — Add products and set discounts
   - **Advanced** — Configure style and settings
5. Enter an **Offer name** in the right sidebar
6. Click **Save**

## Offer status

| Status | Description |
|--------|-------------|
| **Draft** | Saved but not shown at checkout |
| **Active** | Live at checkout (within scheduled date range) |
| **Scheduled** | Will become active on the set start date |
| **Expired** | Past the end date — no longer shown |

## Warning banner

If the checkout block is not activated or your store is not on Shopify Plus, a yellow warning banner appears at the top of the editor. Follow the link in the banner to fix the issue before the offer can go live.

https://prnt.sc/miTre3s_3Z7e

## Edit or delete an offer

From the Offers list, use the action icons on the right of each row:

- **Edit icon (pencil)** — Open the offer editor
- **Delete icon (trash)** — Permanently remove the offer and its associated discount
