# Customize Style & Settings

Below Step 2 in the offer editor, you can customize how the upsell block looks and behaves.

## Style tab

| Setting | Description | Default |
|---------|-------------|---------|
| **Title** | Heading shown at the top of the upsell block | "You may also like" |
| **Description** | Subtext shown below the title | "Recommended for you" |
| **Show quantity selector** | Lets buyers choose how many units to add (1–10) | On |

https://prnt.sc/6efxKn4bklrh

## Settings tab

### Hide items already in order

When enabled, products that the buyer has already added to their cart will not appear in the upsell block.

### Discount combinations

Control whether upsell discounts can stack with other active discounts on the order:

| Option | Description |
|--------|-------------|
| **Product discounts** | Allow stacking with product-level discounts |
| **Order discounts** | Allow stacking with order-level discount codes |
| **Shipping discounts** | Allow stacking with free shipping discounts |

### Active date

Set the time period during which the offer is active:

- **Start date** — Required. The offer becomes active at this time.
- **End date** — Optional. Enable the checkbox to set an expiry. The offer automatically deactivates when the end date passes.

https://prnt.sc/_XE9tebLKJEY

{% hint style="info" %}
If no end date is set, the offer stays active indefinitely until you manually deactivate it.
{% endhint %}
