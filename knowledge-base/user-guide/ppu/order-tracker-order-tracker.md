# Order Tracker

Monitor all orders where upsell offers were shown, accepted, or declined.

## Quick Report

The Order Tracker dashboard shows a summary of your upsell activity:

* Total orders with upsell offers shown
* Accepted offers count and revenue
* Declined offers count

## Order details

For each order, the tracker shows:

| Field                | Description                                                      |
| -------------------- | ---------------------------------------------------------------- |
| **Order number**     | Shopify order reference                                          |
| **Offer type**       | Post-purchase (PPU), Thank you page (TKU), or Order status (OSU) |
| **Status**           | Shown, Accepted, or Declined                                     |
| **Products claimed** | Which upsell products the customer accepted                      |
| **Revenue**          | Upsell revenue from this order                                   |
| **Payment status**   | Whether payment was completed                                    |
| **Date**             | When the order was placed                                        |

## Filtering and pagination

* Filter orders by **offer type** (PPU, TKU, OSU)
* Filter by **status** (Shown, Accepted, Declined)
* Paginate through results for large order volumes

## Use cases

* **Verify test orders** — After placing a test order, check the Order Tracker to confirm your offer was shown and tracked correctly
* **Monitor daily performance** — Quick view of how many customers accepted or declined offers
* **Investigate issues** — If a customer reports a problem with an upsell, find the specific order to review what happened
* **Track unpaid items** — Identify any upsell orders where payment wasn't completed
