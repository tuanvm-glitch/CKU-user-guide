# Design

### How to set up Cart Drawer Design:&#x20;

{% stepper %}
{% step %}
### Choose a preset&#x20;

**Presets**\
Select a quick theme from preset color schemes to instantly apply coordinated color styles.

**Auto-match with your theme**

Click “Auto-match theme” to preview color sets that match your store’s existing design.

{% hint style="info" %}
If this feature is not active, click “Contact us” to request theme integration.
{% endhint %}
{% endstep %}

{% step %}
### Set up Background colors

Customize the background colors of different parts of the cart:

* **Header**
* **Body**&#x20;
* **Footer**
{% endstep %}

{% step %}
### Set up Button Styles

* **Background color**\
  Set the primary color for the checkout/apply buttons (e.g. `#B42F2F`)
* **Text color on button**\
  Choose contrasting font color for clarity (e.g. `#FFFFFF`)
* **Gradient background** _(optional toggle)_\
  Adds a gradient to buttons for a modern touch.
{% endstep %}

{% step %}
### Set up General Styling

Customize the general colors of different parts of the cart:

* **Text color**&#x20;
* **Icon color**&#x20;
* **Divider & delete icon color**&#x20;
{% endstep %}

{% step %}
### Set up Price & Discount Colors

Control how pricing and promotional values appear:

* **Price color:** `#B42F2F`
* **Compare price color (strikethrough):** `#8D8A8A`
* **Saving color:** `#00C154` (e.g. “Save $8”)
* **Background discount:** `#FFCDCD`
* **Text & icon on discount:** `#B42F2F`

These options help you highlight value and increase perceived savings clarity.
{% endstep %}
{% endstepper %}
