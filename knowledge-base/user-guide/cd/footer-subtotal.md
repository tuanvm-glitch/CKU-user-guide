# Subtotal

### How to set up the Subtotal

{% stepper %}
{% step %}
### Navigate the Subtotal&#x20;

Navigate to **Footer > Subtotal** in the Cart Drawer editor to configure this feature.
{% endstep %}

{% step %}
### Adjust the Settings&#x20;

**Label:** Define the title shown before the subtotal amount.\
&#xNAN;_&#x45;xample:_ `Subtotal`

**Show compare-at price:** Toggle this option ON to show the original (strikethrough) price next to the discounted subtotal. It helps reinforce discount-based urgency and perceived value.
{% endstep %}

{% step %}
### Customize the Display

**Font weight** Choose between:

* `Normal` – for a minimalist feel
* `Bold` – to make the subtotal stand out

**Font size**\
Adjustable via slider (8px to 24px), giving you control over visual hierarchy and emphasis.
{% endstep %}
{% endstepper %}
