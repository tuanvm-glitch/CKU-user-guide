# Trust badges

### How to set up the Trust Badges

{% stepper %}
{% step %}
**Navigate the Trust Badges**

Navigate to **Footer > Trust badges** in the Cart Drawer editor. Toggle the feature on to begin customizing.
{% endstep %}

{% step %}
**Icon Selection**

Choose the payment icons you'd like to display from a wide library of options. Supported icons include (but are not limited to):

* Visa, Mastercard, PayPal, AMEX, Apple Pay, Google Pay, Stripe, Klarna, Afterpay, Bitcoin, and more.
{% endstep %}

{% step %}
**Icon Size**

Select how large the trust icons appear in the cart drawer:

* **Small**
* **Medium** _(default)_
* **Large**

Pick a size that balances visibility with design harmony.
{% endstep %}
{% endstepper %}
