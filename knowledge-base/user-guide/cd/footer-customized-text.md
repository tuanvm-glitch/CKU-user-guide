# Customized text

### How  to set up the Customized Text feature:

{% stepper %}
{% step %}
### Navigate the Customized Text

Go to **Footer > Customized text** in the Cart Drawer editor and toggle the feature ON.
{% endstep %}

{% step %}
### Customize the Settings&#x20;

Use the rich text editor to input any message you want customers to see before checkout.

**Use cases**&#x20;

* “Tax included. Shipping calculated at checkout.”
* “By placing your order, you agree to our return policy.”
* “Orders will be processed within 24 hours.”
{% endstep %}

{% step %}
### Set up the Checkbox for customer confirmation

Enable this option to **add a checkbox before the custom text**. Customers must tick this checkbox to proceed with checkout, making it perfect for legal or policy confirmations.

**Use cases**

* Terms & Conditions agreement
* Subscription or refund policy confirmation
* Legal compliance for specific markets (e.g. GDPR)

**Error messages**&#x20;

Edit the message that will appear if customers try to checkout without ticking the checkbox.&#x20;
{% endstep %}
{% endstepper %}
