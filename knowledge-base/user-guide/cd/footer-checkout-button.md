# Checkout button

🛍️ Checkout Button

This button drives action. Customizing its label and icon helps reinforce the next step and keeps it visually aligned with your brand.

* **Label**: Customize the button text and use dynamic variables like `{{total_price}}` (e.g., “Checkout $45.00”).
* **Icon**: Choose the icon shown next to the button (cart, lock, bag, etc.).
* **Direct to cart page**: (Optional) Skip the drawer and go directly to Shopify’s cart page when clicked.
