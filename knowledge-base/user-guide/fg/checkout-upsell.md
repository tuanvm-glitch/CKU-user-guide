# Checkout Upsell

Show upsell product suggestions at checkout to increase Average Order Value (AOV). Offer targeted product recommendations with discounts directly on the checkout page.

* Add product upsells on the **checkout page**
* Let customers add extra items with one click
* Supports fixed or percentage discounts
* **Only available for Shopify Plus merchants**

### **1. Where to find it in the app**

* Go to **AOV Free Gift app → Campaign list → Create new campaign**
* Select campaign type: **Checkout Upsell**
* Click **Create** under **Checkout upsell**

### **2. How to set up in Admin**

*   **Step 1: General settings**

    Enter Campaign name (e.g. “T-Shirt Upsell”)\
    Name your Discount code (shown in cart)

<figure><img src="https://1289686769-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMo81Ak7Ig0Z6bLRmWpSA%2Fuploads%2FV9dnhJAzVEXwYR0uP24V%2Fcp.png?alt=media&#x26;token=e51ccae2-38e6-47a3-a2d4-43504c1f4ce1" alt=""><figcaption></figcaption></figure>

*   **Step 2: Set trigger conditions**

    Choose one of:

    * All products
    * Specific products
    * Specific collections

    _(Expected result: Upsell block only shows when these triggers are met)_
*   **Step 3: Choose upsell product**

    Click **Browse** to select 1 product\
    Choose discount type:

    * Percentage (% off)
    * Fixed amount ($ off)\
      → Set discount value\
      → Select discount combination:
*   **Step 4: Save and continue**

    Click **Save** to confirm

Make sure saving campaign before you go to next step "Add block to theme". Because once you save campaign successfully, the checkout block would appear on Shopify theme settings for you to add

### **5. How it works on Storefront**

> Customer experience on checkout page.

*   **Step 1: Customer adds a trigger product**

    → Checkout page loads with upsell widget (based on trigger)\
    → Widget shows suggested product with discount

    <figure><img src="https://1289686769-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMo81Ak7Ig0Z6bLRmWpSA%2Fuploads%2FfHcRaRYzrSFN57dEoHBm%2Fimage.png?alt=media&#x26;token=a8cd6ecf-9147-42b8-a5cc-d118e749e4ef" alt=""><figcaption></figcaption></figure>
* **Step 2: Customer interacts with widget**\
  → Clicks **Add** to insert upsell item into order summary
*   **Step 3: Product is added with discount**

    → Discount is applied automatically\
    → Upsell product shows in order summary before payment

### **6. FAQs & Troubleshooting**

**Q1. The upsell widget is not visible.**

* Make sure you added the “AOV Checkout Upsell” block in your **Checkout theme**
* Confirm the trigger condition is matched by products in cart

**Q2. The discount isn’t applying.**

* Check if the discount type and value are set correctly
* Make sure the upsell product is **published** and in stock

**Q3. Why don't I can see the AOV block to add to Shopify theme settings**

* You must save campaign successfully to see and add AOV block to theme
