# Customer location

### Applicable Campaign Types

* Gift with cart value
* Gift with collection value
* Buy X Get Y
* Gift with quantity purchase
* Order goal
* Shipping goal

***

### How it Works

* Merchants can select one or multiple countries where the campaign will apply.
* Only customers whose shipping address is located in the selected country/region will be eligible for the campaign.
* Customers outside the selected locations will not see or receive the campaign benefits.

***

### Example

* Merchant sets up a **“Buy X Get Y”** campaign.
* Customer eligibility is set to **Customer location**.
* Selected locations: **Algeria, China**.
* Result: Only customers from Algeria and China can claim the free gift.

***

### Setup Instructions

1. Create or edit a campaign.
2. Go to **Advanced settings > Customer eligibility**.
3. Select **Customer location**.
4. Search and select the country or region(s) you want to target.
5. Save and activate the campaign.

### Best Use Cases

* **Regional promotions:** Run campaigns during local holidays or events in a specific country.
* **Shipping availability:** Only offer free gifts to regions where shipping is supported.
* **Market testing:** Launch promotions in selected markets before expanding globally.
