# Tiered Reward Box

{% hint style="success" %}
Tiered Reward Box can be applied for 3 campaign types: Gift With Cart Value, Gift With Collection Value and Gift With Quantity Purchase.
{% endhint %}

### What is Tiered Reward Box?

The Tiered Reward Box is a widget that appears as a floating element on your store, providing a visually appealing way to highlight ongoing promotions. Specifically, it focuses on tiered gift campaigns—promotions where customers unlock rewards based on their spending levels. The more they spend, the more they receive, creating a compelling incentive for higher cart values.

### How to Use Tiered Reward Box

#### Step 1: Access Tiered Reward Box

Navigate **Tiered Reward Box** section in the app's dashboard.

Toggle the **"Tiered Reward Box"** button on to enable the widget.

#### Step 2: Configure Trigger Campaign

The Tiered Reward box is applied for 1 among 3 available campaign types, which are **Gift with Cart Value, Gift with Collection Value and Gift with Quantity Purchase.**

**Select campaign type**

Choose which campaign type among the three above-mentioned options that the gift will be triggered to create your tier reward structure.

<figure><img src="https://1289686769-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMo81Ak7Ig0Z6bLRmWpSA%2Fuploads%2FdaTwQ0nAChUKAMA8KmFL%2Fimage.png?alt=media&#x26;token=2d9c0db6-6d64-4a1f-aceb-f8b920d62bc9" alt=""><figcaption></figcaption></figure>

**Select campaign**

Among campaign(s) which you have set up in that selected type, choose the one(s) that you want to apply in the tier reward box.

<figure><img src="https://1289686769-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMo81Ak7Ig0Z6bLRmWpSA%2Fuploads%2Fs2xfFeQpnQLMnotWQBpF%2Fimage.png?alt=media&#x26;token=611fbcc1-b04a-4e54-a95d-3cdc6310f84d" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
It's recommended to select at least 3 campaigns to create various spending threshold to keep customers engaged.
{% endhint %}

#### Step 3: Customize the Setting

**Gift box content**

Set up messages appearing on the box to ensure effective communication with your customers.

* **Header**: Add an engaging header, e.g., "Free Gift Milestone."
* **Description**: Provide a brief message to inform customers of the rewards they can unlock by reaching each milestone.
* **Promotion Message**: Use dynamic tags to show progress (e.g., “Spend \{{trigger\_amount\}} to get \{{gift\_number\}} free gift(s)”).
* **Progress Message**: Customize text to display remaining amount for the next reward (e.g., “Spend \{{progress\_amount\}} more to get \{{gift\_number\}} free gift(s)”).
*   **Gift box icon**

    Select an **icon** that resonates with your brand. This will display in the reward box and enhance the visual appeal.

{% embed url="https://app.arcade.software/share/DXXLH2jpCtx7WienZmOx" %}

#### Step 4: Set up Gift box behavior

Decide when the gift box should appear to your customers.

If you want the Tiered Reward Box to open as soon as customers land on your store, tick the box.

If you want to control how long the Tiered Reward Box takes to load, set the number of second by filling in the box.

Finally, if you want to prompt the box whenever a qualifying item is added, enable “Open every time a customer adds an item”.

#### Step 5: Customize Widget theme

#### In the **Widget Theme** section, you can select between regular themes (like _Winter_ and _Stone_) or seasonal themes (such as _Halloween_) for your own purposes.

<figure><img src="https://1289686769-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMo81Ak7Ig0Z6bLRmWpSA%2Fuploads%2FfwzDpnd7XA5Qr1F1d3Cd%2Fimage.png?alt=media&#x26;token=602a2e59-aabb-4c9d-9dda-673e41e19830" alt=""><figcaption></figcaption></figure>

#### Step 6: Set up Widget style

**Set up Tiered Reward Box Background**

Choose whether you want the gradient banner background or not by ticking in the box.

Adjust the background color to harmonize with your brand’s color palette.

<figure><img src="https://1289686769-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMo81Ak7Ig0Z6bLRmWpSA%2Fuploads%2FOFEJJKwYnaSGtcTmVd9m%2Fimage.png?alt=media&#x26;token=b7a1a039-525c-4b3b-86bc-b6187edce4f1" alt=""><figcaption></figcaption></figure>

**Set up Circular Progress**

The Circular Progress indicator is a great way to motivate customers by showing how close they are to reaching the next reward tier. To customize it:

**Foreground Color**: Choose a color for the portion of the circle that represents the progress made toward a milestone.

**Background Color**: Select a color for the remainder of the circle that represents the remaining amount needed to reach the milestone

<figure><img src="https://1289686769-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMo81Ak7Ig0Z6bLRmWpSA%2Fuploads%2Fj2AnWQELur9bkdQfFws2%2Fimage.png?alt=media&#x26;token=c074cde5-7c56-4b9c-a877-a188c7c2b770" alt=""><figcaption></figcaption></figure>

**Set up Text Color, Icon, Disable Color, Border Color**

Change the colors you wish to match the overall campaign.

<figure><img src="https://1289686769-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMo81Ak7Ig0Z6bLRmWpSA%2Fuploads%2FdHsNtrIaDxzxgUwkIiTV%2Fimage.png?alt=media&#x26;token=1f8c21bd-0833-417f-9508-84245a63130d" alt=""><figcaption></figcaption></figure>

#### Step 6: Finalize the Setting

Preview the Tiered Reward Box by seeing how it will appear to customers on the right sight of the settings.

After verifying that everything works as expected, click **Save**.

Once you’re satisfied with the setup, go live and watch your average order value rise as customers eagerly work toward earning free gifts!
