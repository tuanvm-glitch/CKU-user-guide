# How to set up Order Goal campaign

### What is Order Goal?

Order Goal is a powerful feature in AOV Free Gift app that enables you to incentivize larger purchases by offering tiered discounts based on the customer's total order value.

### Why to use Order Goal?

<details>

<summary>Increase Average Order Value (AOV)</summary>

This feature encourages customers to spend more to receive attractive discounts, thus increasing the AOV.

</details>

<details>

<summary>Delight Customers</summary>

Order Goal provides customers with a rewarding shopping experience by clearly showing them their savings or next rewards, making the purchases more engaging.

</details>

<details>

<summary>Boost Sales Conversion</summary>

Discounts tied to specific cart values can reduce cart abandonment and increase conversion rates.

</details>

### How to Set up Order Goal

#### Step 1: Choose Order Goal campaign

Navigate the **Campaigns section** in the Menu > click **Create campaign** on the top right corner.

Choose Order Goal and click Select button to start setting the campaign. You can also click Explore use case to see if there is any provided template that suits your demand.

{% embed url="https://app.arcade.software/share/13TvQDzoL9GjyU3RU7a8" %}

#### Step 2: Set up General Information

**Campaign name**

Enter a name for this campaign. The name is only use for internal purposes only, meaning that your customers will not see it in the storefront.

**Discount code**

Set a code that will apply for this offer. Your customers will see it in their cart once they meet the condition of the campaign.

{% embed url="https://app.arcade.software/share/RjJq1GJU8Jg3KIR01w2K" %}

#### Step 3: Set up Campaign trigger

**Minimum purchase amount**

Set the **Minimum Purchase Amount** to define the cart threshold that customers need to qualify for the discount.

Example: Set the minimum purchase value to $50 to trigger the discount.

#### Step 4: Set up Campaign Offer

**Discount settings**

Determine the type and value of the discounts your customers will receive when they meet the campaign's cart value goal.

We offer two **discount types** for you to choose:

* Percentage
* Fixed amount

Then, enter the discount value of that type. For instance:

* Percentage: 10% off
* Fixed Amount: $5 off

**Discount combinations**

Specify if this discount can stack with product, order, or shipping discounts to increase customer benefits.

{% embed url="https://app.arcade.software/share/g0LhxXpGPPl4OI0RoZOw" %}

#### Step 5: Customize the Campaign Appearance

At the moment, the gift selection method is default as Automatic with a Congrats bar widget, which means discounts will automatically apply when customers qualify. Customers will be notified by a bar saying a congratulation message on their achievements.

**Content**

You can edit the Content to personalize the message, such as: "Cheers to savings, your order gets 10% OFF!"

Use dynamic placeholders like `{{discount_value}}` to customize for each discount tier.

#### Tiered reward box

The **Tiered Reward Box** is a powerful new way to showcase the Order Goal campaign directly in your storefront. This widget visually tracks customer progress toward unlocking rewards, making it easier to encourage higher order values. By displaying tiered rewards in an engaging format, merchants can create a more dynamic and motivating shopping experience.

{% hint style="warning" %}
The Tiered Reward Box feature is available exclusively for paid users.
{% endhint %}

Click **Set-up widget** to configure the widget.

You will be redirected to the **Tiered Reward Box settings** page, where you can customize the design and reward structure. Read this guide to see how to configure the Tier reward box.

#### Announcement bar

If you want to display a fixed announcement bar on the header, ensuring customers are notified of their discount progress, we highly recommend you to installing Avada Boost Sales app.

Read this guide to better understand how to set up an Announcement bar as well as other seamless ways to promote the Order Goal campaign.

#### Step 6 (Optional): Schedule the Campaign

If you want your promotion to run for a specific period of time, you can totally set the start and end date/ time. Notice that timezone is based on (GNT - 5:00) America/ New York.

**Start date**

Add the start date and time when your promotion will begin.

**End date (Optional)**

Add the end date and time when your promotion will stop.

Review all the settings again and save the campaign once you feel satisfied.
