# How to set up

### What is Frequently Bought Together?&#x20;

Being popularized by by Amazon, Frequently Bought Together features lets you showcase complementary products that customers are likely to by together. However, rather than just products that are bought together, we often see it used specially for upsells or cross-sells.&#x20;

### Why Frequently Bought Together?&#x20;

<details>

<summary>Maximize Revenue and Profits</summary>

By reminding customers of complementary products they might need, Frequently Bought Together recommendations nudge them to spend more per purchase, thus boosting your revenue and profits.&#x20;

</details>

<details>

<summary>Time-saving Product Discovery </summary>

Frequently Bought Together saves customers valuable time and efforts of searching for complementary items, since it allows them to complete their purchase with everything in one go.&#x20;

</details>

<details>

<summary>Data-driven and AI-generated Upsells</summary>

Based on real customer behaviors and AI generation, Frequently Bought Together ensures you are suggesting products that customers are more likely to purchase, making more effective upsells.

</details>

### Requirements before using

* Products must already be added to your Shopify store.
* You must have the **AOV.ai Bundle app installed and active**.
* (Optional) Make sure you know which products are commonly purchased together (you can use AI Recommendation if unsure).

### How to Set up?

#### Step 1: Set up the offer

{% stepper %}
{% step %}
Go to **AOV.ai Bundle > Offers > Create > Frequently Bought Together.**
{% endstep %}

{% step %}
Enter an Offer name. This name is internal only.
{% endstep %}

{% step %}
In **Customer meets condition**, choose **one:**

* **Specific products** → click **Select**, choose one or more products that will _trigger_ the offer on their product pages.
* **Specific collection** → click **Select**, choose one or more collections; any product in those collections will _trigger_ the offer.
{% endstep %}

{% step %}
Choose **Suggested products** to pair with the trigger:

* **AI Recommendation** – auto-suggested by AOV.ai based on patterns.
* **Same collection** – items from the same collection.
* **Specific product** – manually pick a product.
* **Specific collection** – manually pick a collection.
{% endstep %}

{% step %}
(Optional) Enable **Add free gift** if you want to include an additional free item.
{% endstep %}
{% endstepper %}

#### Step 2: Configure discount settings

This section controls **how much customers save** when they buy the bundle. You can set the discount type, value, and any rules for how the discount applies.

{% stepper %}
{% step %}
Toggle **Discount Bundle** on.
{% endstep %}

{% step %}
Select a **Discount type**:

* Percentage (e.g., 10% off)
* Fixed amount
{% endstep %}

{% step %}
Enter the discount value.
{% endstep %}

{% step %}
(Optional) Configure:

* **Discount combinations** (e.g., allow stacking with product or order discounts).
* **Maximum discount uses:** Limit how often the discount can be used:
  * **Limit number of times this discount can be used in total** – set a global usage cap across all customers (e.g., first 50 uses only).
  * **Limit to one use per customer** – each customer can use the bundle discount only once.
{% endstep %}
{% endstepper %}

#### Step 3: Advanced configuration

These options refine how the bundle behaves. Toggle to enable the feature(s) you want.

{% stepper %}
{% step %}
Apply minimum product to receive a discount

* Require customers to add a certain number of items before the discount applies.
* Example: Must add at least 3 products from the bundle to unlock the discount.
{% endstep %}

{% step %}
Exclude trigger product from discount

* The trigger product will not receive any discount.
* Only the suggested products get discounted.
* Example: Polo Shirt (trigger) stays at full price, but Sneakers and T-Shirt get 10% off.
{% endstep %}
{% endstepper %}

#### Step 4: Schedule (optional)

Define when the bundle discount should start and end.

Useful for campaigns tied to events like Back-to-School or Black Friday.

#### Step 5: Preview and Save the Offer

Review the **Preview** on the right panel to see how the widget will look on your store’s product page.

Once satisfied, click **Save** to activate the offer.

That's it! Leverage the **Frequently Bought Together** feature to give your customers value-packed deals while boosting your revenue. Start creating your first bundle today! 🚀
