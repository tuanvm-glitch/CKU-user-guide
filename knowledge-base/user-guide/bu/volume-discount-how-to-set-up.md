# How to set up

### What is Volume Discount campaign?&#x20;

Volume Discount allows you to set up discounts that lower the price of the items bought when your customers increase quantities they purchase. It encourages larger purchases and gives your customers a reason to add more items to their cart while maximizing your revenue.

### Why to use Volume Discount

<details>

<summary><strong>Increase average order value (AOV)</strong></summary>

Encourage customers to add more items by rewarding them with discounts as their cart grows.

</details>

<details>

<summary><strong>Move more inventory</strong></summary>

A great way to sell through seasonal or overstocked products.

</details>

<details>

<summary>Flexible promotion style</summary>

Choose between percentage or fixed discounts, add free gifts, or highlight popular tiers with badges.

</details>

<details>

<summary><strong>Reward loyal buyers</strong></summary>

Give better deals to shoppers who purchase multiple quantities.

</details>

### How to set up Volume Discount campaign?&#x20;

#### Step 1: General settings

In the dashboard of your app, choose **Create campaign** > **Volume Discount** from the list of campaign types to start create one.

**Offer name**: Enter an internal name for your discount campaign. This is only for you to identify the offer.

#### **Step 2: Set up** Trigger conditions

Decide which items trigger the discount:

* **All products**: discount applies storewide.
* **Specific products**: select individual items.
* **Specific collections**: apply to selected collections.\
  You can also exclude products or collections if needed.
* **Markets**: choose where the discount is active (e.g., All markets, US only).

#### **Step 3:** Discount type

{% stepper %}
{% step %}
Choose how the discount is calculated:

* **Percentage** (e.g., 10% off).
* **Fixed amount** (e.g., $5 off).
{% endstep %}

{% step %}
Define the Discount tiers

Tiers let you reward customers with bigger discounts as they buy more:

* **Product quantity**: how many items must be purchased to unlock the tier.
* **Discount value**: the discount given at that tier.
* **Title and Subtitle**: customize how the offer is described to customers.
* **Enable badge**: highlight a tier with a badge like “Best deal.”

Example setup:

* Tier 1 → Buy 2 items, get 10% off.
* Tier 2 → Buy 5 items, get 20% off.
{% endstep %}

{% step %}
(Optional) Attach a gift

This option lets you **add a free product** as part of a discount tier. When customers reach the required quantity, the selected product is added automatically to their cart.

* **Select product**: choose which product will be given as a gift.
* **Discount type**:
  * _Free_: the gift is completely free.
  * _Discounted_: apply a specific discount instead of giving it free (if available in your setup).
* **Content**: text shown beside the gift in the widget (example: _+ Free gift_).

Example: If Tier 2 is “Buy 3, Get 15% Off,” you could attach a branded tote bag as a free gift when the discount unlocks.
{% endstep %}

{% step %}
#### Display custom image

This option allows you to **add an image to visually highlight a discount tier**. It makes your promotion more attractive and easier for shoppers to notice.

* **Add images**: upload a `.gif`, `.jpg`, `.jpeg`, or `.png` file (up to 5MB).
* **Size slider**: adjust how large the image appears in the widget.
* **Corner slider**: control how rounded the image corners look.

Example: You could add an image of a gift product, a “Most Popular” badge, or a seasonal promo banner to make the tier stand out.
{% endstep %}
{% endstepper %}

#### Step 4: Advanced configuration

These settings let you control how your Volume Discount behaves in more detail.

{% stepper %}
{% step %}
**Discount combinations**

Choose whether this discount can be combined with other types of discounts:

* **Product discounts**: Allow customers to use product-specific discounts (e.g., a T-shirt already 20% off) together with the volume discount.
* **Order discounts**: Let customers stack order-wide discounts (e.g., “10% off entire cart”) with the volume discount.
* **Shipping discounts**: Let customers also apply free shipping or reduced shipping discounts.

{% hint style="success" %}
Best practice: Enable stacking only if you want to run aggressive promotions. If not, leave unchecked to protect margins.
{% endhint %}
{% endstep %}

{% step %}
**Pricing options**

Decide how prices and savings appear to customers:

* **Show prices per item**: Display the adjusted price per unit (e.g., “$9.00 each when you buy 5”). Helps customers see the deal clearly.
* **Hide option’s compare-at price**: Remove the original crossed-out price if you don’t want to show a side-by-side comparison.
* **Enable display of savings amount by discount tier**: Show how much money or percentage customers save at each tier (e.g., “Save 15% when you buy 3”).
* **Apply discount for next quantity only**: Discount applies only to items above the threshold. Example: If Tier 1 = Buy 2 get 10% off, only the 3rd item onward gets discounted.
* **Price rounding**: Round discounted prices to cleaner numbers (e.g., $9.97 → $10.00).

{% hint style="success" %}
Best practice: Use **Show prices per item** if you want to make bulk discounts more persuasive.
{% endhint %}
{% endstep %}

{% step %}
**Maximum discount uses**

Control how often the discount can be redeemed:

* **Limit number of times this discount can be used in total**: Set a global cap (e.g., only first 100 customers can claim).
* **Limit to one use per customer**: Each customer can only use the discount once, regardless of how many orders they place.
{% endstep %}

{% step %}
**Customer eligibility**

Choose who can access the discount:

* **All customers**: Everyone sees and can use the discount.
* **Specific customer segment**: Restrict the discount to groups (e.g., VIP customers, wholesale buyers, or tagged customer lists).

{% hint style="success" %}
Best practice: Use eligibility rules to target your best customers with exclusive deals.
{% endhint %}
{% endstep %}
{% endstepper %}

#### Step 5 (Optional) **Schedule**

Set the start date and (optional) end date.

The campaign runs automatically based on your store’s timezone.

#### Step 6: Preview and Save&#x20;

After configuring all settings, review your campaign settings to ensure everything is correct.&#x20;

Once satisfied, click **Save** at the bottom of the page to launch your Volume Discount campaign.

And that's it! You've successfully set up the Volume Discount campaign and create a captivating experience for your customers. Happy selling!
