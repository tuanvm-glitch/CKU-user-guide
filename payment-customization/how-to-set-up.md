# Payment Customization

{% hint style="success" %}
**Available on all Shopify plans.** No Shopify Plus required.
{% endhint %}

Payment Customization lets you control which payment methods are shown at checkout and how they appear — powered by Shopify Functions. Changes take effect immediately after saving a rule.

## What you can do

| Rule | Description |
|------|-------------|
| **Hide** | Remove one or more payment methods from checkout |
| **Rename** | Change the display name of a payment method |
| **Reorder** | Change the order in which payment methods appear |

All rule types support trigger conditions — so rules can apply only when specific cart or customer criteria are met.

## Create a rule

1. Go to **Checkout Customization** in the left menu → **Payment** tab
2. Click **Create rule**
3. Choose a rule type: Hide, Rename, or Reorder
4. Set a **Rule name** and **Status** (Active / Inactive)
5. Configure conditions and rule action
6. Click **Save**

The rule takes effect at checkout immediately.

## Manage existing rules

From the Checkout Rules list, you can:

- **Toggle status** — Activate or deactivate a rule without deleting it
- **Edit** — Change conditions or actions
- **Delete** — Permanently remove the rule
