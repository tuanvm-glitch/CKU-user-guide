# Reorder Payment Methods

Use this rule to change the order in which payment methods appear at checkout.

## When to Use

- Move your preferred payment method (e.g., credit card) to the top
- Promote a specific option (e.g., Buy Now Pay Later) for higher-value orders
- Reduce friction by showing the most common method first

## Setup

1. Go to **Checkout Customization** → **Payment** tab
2. Click **Create rule** → select **Reorder payment method**
3. Set a **Rule name** and **Status** (Active / Inactive)
4. Configure **Trigger Conditions** (optional)
5. In the **Rule Action** section:
   - Type the payment method name and click **Add**
   - Drag and drop the items to set the desired order
   - Methods listed here will appear first; unlisted methods follow in their default order
6. Click **Save**
