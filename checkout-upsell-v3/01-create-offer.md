# Create an Offer

## Step 1: Go to Offers

In the app sidebar, click **Offers** to see all your upsell offers.

![Offers list page](https://i.ibb.co/gbzdr5bK/offers-list-v2.png)

### List page overview

| Feature | Description |
|---------|-------------|
| **Tabs** | Filter by status: All, Active, Scheduled, Expired |
| **Search** | Search offers by name |
| **Sort** | By Priority (asc/desc) or Date Created (oldest/newest) |
| **Bulk actions** | Select multiple → Activate, Deactivate, or Delete |
| **Items per page** | 10, 50, or 100 |

Each offer row shows: **Priority** number, **Name**, **Status** badge (Active/Scheduled/Expired), **Type**, **Views**, **Orders**, **Revenue**, and action icons (edit/delete).

---

## Step 2: Choose offer type

1. Click **Create offer** (top right)
2. Select **Checkout Upsell**
3. Click **Select**

![Offer type selection](https://i.ibb.co/671N8SC3/offer-type-v2.png)

---

## Step 3: Configure the offer

The editor has two columns:
- **Left** — Step 1: Trigger Conditions, Step 2: Offer Products, Step 3: Advanced Settings
- **Right (aside)** — Offer name, Status, App block status, Live preview

![Offer editor](https://i.ibb.co/Txfd7G8R/editor-new-v2.png)

### Aside panel

| Field | Description |
|-------|-------------|
| **Status** | Click badge to change: Active (green) or Expired. Auto-changes with scheduling. |
| **Offer name** | Required. Auto-generated: "Checkout upsell #1", "#2", etc. |
| **App block** | Shows "Added" (green) if checkout block is in your theme, "Not added" otherwise. Click "Go to theme settings" to add it. |

{% hint style="warning" %}
If you see a **yellow warning banner** ("Checkout Upsell block isn't activated"), click **"Go to theme settings"** to add the AOV.AI block to your checkout theme.
{% endhint %}

---

## Step 4: Save

Click **Save** in the top bar. The offer is live when:
- Status is **Active**
- Start date has passed (or no scheduling)
- Checkout block is added to your theme

---

## Offer statuses

| Status | Badge | Meaning |
|--------|-------|---------|
| **Active** | Green | Live at checkout now |
| **Scheduled** | Yellow | Will auto-activate on start date |
| **Expired** | Gray | Past end date — no longer shown |
| **Draft** | Blue | Saved but not activated |
| **Test mode** | Orange | Only shown to test emails |

---

## Edit or delete

From the Offers list:
- **Edit** — click offer name or pencil icon
- **Delete** — click trash icon, or select multiple → bulk Delete
- **Activate/Deactivate** — use action icon or bulk actions
