# Discount List — How to Set Up

{% hint style="warning" %}
**Shopify Plus required.** This feature also requires the AOV.AI block to be activated in your Checkout theme. See the **Installation** page in Getting Started.
{% endhint %}

## What it does

Discount List displays your active discount codes directly at checkout. Buyers can see available discounts and apply them with one click — without needing to remember or copy a code separately.

This reduces friction at checkout and helps buyers feel they are getting the best deal, increasing conversion.

## Activate the Discount List block

1. Go to **Shopify Admin** → **Online Store** → **Themes** → **Customize**
2. Select **Checkout** from the page dropdown
3. In the App blocks panel, find **AOV.AI — Discount List** and drag it to your preferred position in the checkout
4. Click **Save**

## Which discounts are shown

The Discount List pulls from your active Shopify discount codes. Only discounts that are:

- Currently active (within their valid date range)
- Not usage-limited beyond their remaining uses
- Applicable to the buyer's current cart

...will be displayed to the buyer.

{% hint style="info" %}
Discount codes marked as "hidden" or used for internal/affiliate purposes should not be active simultaneously if you don't want them shown to all buyers.
{% endhint %}

## Buyer experience

At checkout, buyers see a list of available discount codes with their descriptions. Clicking a code applies it to the order instantly — the order total updates in real time.
